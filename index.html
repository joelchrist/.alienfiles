Description

A place where I store all my configuration files for my systems. I will try to keep these as up-to-date as possible. There are branches depending on different configurations. The mac_osx branch will, as expected, contain configuration files for my OSX setup.

Feedback

If you feel like there is something in these files that shouldn't be there or I could do differently feel free to create an issue. For questions you can always email me.
